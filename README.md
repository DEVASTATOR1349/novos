# Новостройки КМВ и Ставрополя

База проектов новостроек (ЖК) по Ставропольскому краю и Кавказским Минеральным Водам.

- **Ставрополь:** 25 проектов
- **КМВ:** 29 проектов (Пятигорск, Ессентуки, Кисловодск, Железноводск, Лермонтов, Георгиевск, Предгорный округ)
- **Колонки 32–41 (УТП, преимущества, риски, ипотека, доходность):** заполнены для всех

Источник данных: Яндекс Недвижимость (май 2026), рыночные знания.
